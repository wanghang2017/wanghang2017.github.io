    小王 2020-12-24 javaspringboot

    # springboot

    # 特性

    1. 能够快速创建基于 spring 的应用程序
    2. 能够直接使用java main 方法启动内嵌的Tomcat服务器运行Spring Boot程序，不需要部署war文件
    3. 提供约定的starter POM来简化Maven配置 让maven的配置变得简单
    4. 自动化配置，根据项目的maven依赖配置，springboot自动配置spring、spring mvc等
    5. 提供了程序的健康检查等功能
    6. 基本可以完全不使用xml配置文件 采用注解配置

    # spring boot 四大核心

    1. 自动配置
    2. 起步依赖
    3. Actuator （健康检测 用不到）
    4. 命令行界面 （用不到）

    6. springmvc常用的注解

      @Controller

      @ResponseBody

      @RestController = @Controller修饰类 + @ResponseBody修饰类里面所有方法

      @RequestMapping(value="",method) 各种请求都接受

    7. 在springboot中使用事务

      sql语句分为：

      DDL 数据定义语言（alter drop create）

      DML数据操纵语言 (update insert delete) 此时会用到事务

      DQL 数据查询语言 (select)

      DCL 数据控制语言(grant reovke)

      TCL 事务控制语言(commit rollback)

      @Transactional 修饰方法 即可开启事务

      @EnableTransactionManagement 修饰入口类 （可加可不加）

    @Autowired
    private RedisTemplate<Object,Object> redisTemplate;
    @Override
    public Student queryStudentById(Integer id) {
        return studentMapper.queryStudentById(id);
    }
    @Override
    public void put(String key, String value){
        redisTemplate.opsForValue().set(key,value);
    }
    @Override

        String value = (String) redisTemplate.opsForValue().get(key);
        return value;
    }
}
